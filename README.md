# OpenGL Projects with Python

This repository features a collection of 2D and 3D computer graphics projects developed using **Python** and **OpenGL**. The projects demonstrate fundamental and advanced graphics programming concepts, ranging from basic shape rendering to complex 3D transformations.

The primary goal of this repository is to showcase how to leverage the **PyOpenGL** library and **GLFW/Pygame** for creating cross-platform graphical applications.

## 🚀 Key Features & Implementations

The projects in this repository cover the following graphics programming topics:

* **Primitive Rendering:** Drawing points, lines, and polygons using legacy and modern OpenGL pipelines.
* **Coordinate Systems:** Implementation of Model, View, and Projection (MVP) matrices.
* **3D Transformations:** Real-time translation, rotation, and scaling of objects.
* **Camera Systems:** First-person and orbital camera implementations for navigating 3D space.
* **Shaders (GLSL):** Introduction to vertex and fragment shaders for custom rendering effects.
* **Input Handling:** Interactivity using keyboard and mouse events via GLFW or Pygame.

## 🛠 Prerequisites & Dependencies

To run these projects, you need to have Python installed along with the following libraries:

- **Python 3.x**
- **PyOpenGL:** The Python interface to OpenGL.
- **GLFW / Pygame:** For window management and input handling.
- **NumPy:** For efficient matrix and vector calculations.

You can install the dependencies using pip:

pip install PyOpenGL PyOpenGL_accelerate glfw numpy pygame
